# Create: Biotech —— 玩家介绍

[English](INTRODUCTION.md) · [中文](INTRODUCTION.zh-CN.md) · [README](../README.zh-CN.md) · [English README](../README.md)

> 一个机器会呼吸的 Create 附属模组。

## 设计理念

Create 给了我们一个由齿轮、传送带和应力构成的世界 —— 优雅地机械化，也优雅地**冰冷**。Create: Biotech 想问的是另一个问题：

> 如果你工厂里的下一个齿轮，不是铁，而是**活的**呢？

史莱姆会滚动，岩浆怪会燃烧，恶魂会漂浮，鱿鱼会喷墨，猫能让波函数坍缩。每一个生物都是潜在的机构 —— 你只需要把它当成零件，而不是害虫。

所以本模组把活体生物直接种进 Create 的动能网络里：不是装饰，也不是另起一套副系统，而是**真正的零件** —— 它们遵守应力、骑乘传送带、在工作盆里被加工、像黄铜漏斗一样接入装置。生物学变成工程学，主世界变成你的供应链。

整个模组建立在三条主线之上：

- **生物即机构。** 小史莱姆不是掉落源，**它本身就是传送带表面**。恶魂不是用来杀的怪物，**它本身就是升降器**。猫不是宠物，**它本身就是一枚量子硬币**。
- **获取即交互。** 大多数零件无法凭空合成。你必须遇见这只生物、捕获它、喂养它、引诱它。工厂从荒野开始。
- **每个活体部件都有死金属做不到的事。** 生物替代品不是换皮 —— 它们有自己的脾气、代价和极限，是纯粹的 Create 机械永远到不了的角落。

## 模组内容

你会建造能做这些事的工厂：

- **用活物运输货物。** 史莱姆带无限循环；岩浆带会烧掉它运的东西；人力带把奔跑的玩家变成应力。
- **加工的不只是物品，还有生物。** 把小史莱姆扔进工作盆，看机械压机像处理任何配料那样处理它。
- **能升、能飘、能漫游。** 给恶魂套上头盔，它就变成你能驾驶的热气球装置。
- **让鱿鱼帮你印书。** 投入一本附魔书，吐出一份副本 —— 墨水由常驻头足类提供。
- **把经验当作素材收割。** 抽取 XP，把它结晶，让它在经验芽块上生长，或者储存成流体。
- **驯服混乱。** 能装下爆炸的苦力怕爆破室；按指令施法的唤魔者附魔室；只在没人看的时候释放红石的薛定谔的猫。
- **请野生动物帮忙。** 自动吸引动物的固定胡萝卜钓竿；用蛛丝编织物件的蜘蛛组装台；能把怪物抱回家的纸箱。

……以及一长串小零件 —— 万向节、骨棘轮、史莱姆离合器、缓冲垫、防爆物品仓 —— 它们存在的意义，就是让那些活着的机器有地方接入。

不需要把这些全部记住。打开 JEI 里的 Create: Biotech 创造模式标签页，点任何一个方块，Ponder 会用动画告诉你它是怎么工作的。本模组的探索方式和 Create 本身一致：**上手摆弄**。

## 开始游玩

1. 安装 Create 6.0.8（Minecraft 1.20.1，Forge 47.1.33）。可选：JEI、Jade。
2. 把 jar 放进你的 `mods/` 文件夹。
3. 去找一只史莱姆。工厂从那里开始。
